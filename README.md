# Todo CRUD App (React + Vite + Tailwind)

A modern Todo application built with React.

## Features
- Create / Read / Update / Delete tasks
- localStorage persistence
- Priority levels (low / medium / high)
- Filter (All / Open / Completed)
- Search tasks
- Responsive UI with Tailwind CSS

## Tech Stack
- React (Vite)
- Tailwind CSS
- localStorage

## Run Locally
npm install
npm run dev

## Build
npm run build
npm run preview
